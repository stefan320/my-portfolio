Dorado blog

This blog was created as a challenge to try one of the most popular JavaScript framework in the industry, 'Vue.js'. I started by spending a few days learning the technology fundamentals. Following that I dedicated some time to plan the project. In terms of functionality the blog fetches data from an API call in 'JSON' format. Every article is then displayed as a summary of 100 characters. When clicking on an article it will be displayed in full and the possibility to add or remove comments is given. Furthermore the blog includes a filter to provide a better user expereience. For this project I also made sure that no browser refreshing or redirects are needed.
